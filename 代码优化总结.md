### 5.2 代码简化
```kotlin
// 优化前
if (score.name.isNotEmpty()) {
    authorTextView.visibility = View.VISIBLE
    authorTextView.text = score.name
} else {
    authorTextView.visibility = View.GONE
}

// 优化后
authorTextView.apply {
    if (score.name.isNotEmpty()) {
        visibility = View.VISIBLE
        text = score.name
    } else {
        visibility = View.GONE
    }
}
```
