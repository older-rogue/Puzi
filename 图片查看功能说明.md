# 图片查看功能实现说明

## 功能概述
在 ScoreDetailActivity 中添加了图片查看功能，支持以下特性：
- ✅ 点击图片放大显示
- ✅ 左右滑动切换图片
- ✅ 底部显示当前页数和总页数
- ✅ 点击图片关闭查看器
- ✅ 双击图片放大缩小（通过 PhotoView 库实现）

## 实现细节

### 1. 新增文件

#### Activity
- `ImageViewerActivity.kt` - 图片查看器主 Activity
  - 全屏显示
  - 使用 ViewPager2 实现左右滑动
  - 底部显示页码指示器

#### Adapter
- `ImageViewerAdapter.kt` - ViewPager2 的适配器
  - 使用 PhotoView 支持双击缩放
  - 点击图片关闭查看器

#### 布局文件
- `activity_image_viewer.xml` - 图片查看器布局
- `item_image_viewer.xml` - 单个图片项布局

### 2. 修改文件

#### ScoreDetailActivity.kt
- 添加 `imageUrls` 列表存储图片 URL
- 为每个图片添加点击监听器
- 新增 `openImageViewer()` 方法启动图片查看器

#### AndroidManifest.xml
- 注册 `ImageViewerActivity`
- 设置全屏主题

#### build.gradle.kts
- 添加 PhotoView 依赖：`com.github.chrisbanes:PhotoView:2.3.0`

#### settings.gradle.kts
- 添加 JitPack 仓库支持

## 使用的库

### PhotoView
- GitHub: https://github.com/Baseflow/PhotoView
- 功能：支持图片缩放、双击放大、手势操作
- 版本：2.3.0

## 使用方法

1. 在 ScoreDetailActivity 中点击任意曲谱图片
2. 进入全屏图片查看模式
3. 左右滑动切换图片
4. 双击图片进行放大/缩小
5. 单击图片退出查看器
6. 底部显示当前页码（如：1/3）

## 构建步骤

1. 同步 Gradle 依赖
```bash
./gradlew build
```

2. 运行应用
```bash
./gradlew installDebug
```

## 注意事项

- PhotoView 库需要通过 JitPack 仓库下载
- 图片查看器使用全屏模式，隐藏系统状态栏和导航栏
- 支持 Android API 30+
